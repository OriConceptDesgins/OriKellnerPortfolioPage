Underwater diving volunteer Project

This is a design for a Diving environmentalist activism project. The landing page would allow the user to Donate, get in contact or transition to another form where they could enlist to become a volunteer diver. The images and color scheme are meant to make it seem both exciting and fun. The project was created using HTML+CSS.
